GitHub Highlights

Many of these programming projects are interactive. Some of my favorites are "Gone Bananas" and "TTNY-Hackathon".
"Gone Bananas" was a collaborative project designed to make an interactive game written in Java. The player can shoot bananas as the monkeys come closer to the player. This game is inspired by Space Invaders.
"TTNY-Hackathon" is a collaborative project written in Python. The purpose of the project is to help students manage their savings and make a budget. The user can make budgets for different categories. They can deposit and withdraw money. When the user tries to withdraw, the system asks them which category they want to withdraw from. If what they withdraw is over their budget, the system warns the user.
